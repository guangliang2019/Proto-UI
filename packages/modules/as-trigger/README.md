# @proto.ui/module-as-trigger

Proto UI module that provides as-trigger capability for adapters.

## Purpose

Provides as-trigger capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-as-trigger@0.0.1
```
