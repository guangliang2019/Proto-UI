# @proto.ui/module-event

Proto UI module that provides event capability for adapters.

## Purpose

Provides event capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-event@0.0.1
```
