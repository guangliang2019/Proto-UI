# @proto.ui/module-state-accessibility

Proto UI module that provides state accessibility capability for adapters.

## Purpose

Provides state accessibility capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-state-accessibility@0.0.1
```
