# @proto.ui/module-props

Proto UI module that provides props capability for adapters.

## Purpose

Provides props capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-props@0.0.1
```
