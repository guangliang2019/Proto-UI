# @proto.ui/module-expose

Proto UI module that provides expose capability for adapters.

## Purpose

Provides expose capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-expose@0.0.1
```
