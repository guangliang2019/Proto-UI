# @proto.ui/module-test-sys

Proto UI module that provides test system capability for adapters.

## Purpose

Provides test system capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-test-sys@0.0.1
```
