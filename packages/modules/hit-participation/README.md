# @proto.ui/module-hit-participation

Proto UI module that provides host-aligned hit-participation capability for adapters.

## Purpose

Provides reliable hit-testing semantics and click participation interpretation for prototype interaction logic.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-hit-participation@0.0.1
```
