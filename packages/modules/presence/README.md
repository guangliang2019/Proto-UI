# @proto.ui/module-presence

Proto UI module that provides structural presence governance for adapters.

## Purpose

Provides soft mount and unmount timing so transition-driven prototypes can keep host elements present until exit work is complete.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-presence@0.0.1
```
