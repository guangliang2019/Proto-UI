# @proto.ui/module-state

Proto UI module that provides state capability for adapters.

## Purpose

Provides state capability to adapters running Proto UI prototypes.

## Package Role

Adapter-facing module package used by the Proto UI runtime and adapter layer.

## Install

```bash
npm install @proto.ui/module-state@0.0.1
```
