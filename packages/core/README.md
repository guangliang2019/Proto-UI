# @proto.ui/core

Proto UI core syntax and protocol primitives.

## Purpose

Provides the core authoring syntax, protocol primitives, and shared contracts used across Proto UI packages.

## Package Role

Core package for defining prototypes, handles, template primitives, and protocol-facing developer APIs.

## Install

```bash
npm install @proto.ui/core@0.0.1
```
