# @proto.ui/types

Shared type definitions for Proto UI packages.

## Purpose

Provides the unified type definitions shared by the rest of the Proto UI package graph.

## Package Role

Type foundation package used by core, runtime, modules, adapters, and prototype libraries.

## Install

```bash
npm install @proto.ui/types@0.0.1
```
