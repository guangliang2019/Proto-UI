---
title: 'Lucide 图标库'
description: '基于 Lucide 的 Proto UI 图标原型库。'
---

`Lucide` 图标库基于 `lucide-static` 生成图标原型。

它遵循 Proto UI 对图标的两种使用形态：

- 当图标被视作独立交互主体时，使用 `asHook`。
- 当图标只是附属视觉结构时，使用 `render` 片段嵌入到其他原型模板。

为了控制体积，每个图标都可以从独立入口按需引入（`@proto.ui/prototypes-lucide/icons/<name>`），不需要一次性加载全量图标。

继续查看[图标列表页](./icons/)，可直接复制每个图标对应的原型代码。
