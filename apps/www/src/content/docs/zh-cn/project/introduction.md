## 这篇文章要回答什么？

- Proto UI 作为一个项目，究竟是什么。
- 它以什么方式公开、维护和演进。
- 它在整个文档站里承担什么位置。

## Proto UI 是什么？

- 一个探索人机交互协议化表达的开源项目。
- 它关注的是组件交互语义如何独立于具体宿主实现而存在。
- 它不是某个单一框架下的新组件库，而是一套更上层的协议化尝试。

## 项目属性

- 开源协议：MIT
- 项目立场：非营利性
- 当前阶段：以协议语义闭环与可验证实现为主
