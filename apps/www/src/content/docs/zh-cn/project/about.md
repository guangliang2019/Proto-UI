# Proto UI

## Proto UI 是什么？

Proto UI 是一个面向多种运行环境的组件交互协议。它尝试将组件行为定义为一种可以被不同 Runtime 翻译和实现的中立形式。

---

## 项目信息

- 仓库地址：https://github.com/Proto-UI/Proto-UI
- 开源协议：MIT
- 维护者：Proto UI Team

---

## 项目范围

Proto UI 关注的是 UI 组件交互语义的定义、表达与跨宿主翻译。  
它既包括原型本身，也包括将原型映射到不同 Runtime 与宿主能力的适配层设计。

Proto UI 不预设唯一的渲染实现方式，但会为适配器生态提供必要的约束、参考与指导。

---

## 当前阶段

预发布阶段（v0）

核心交互语义仍在持续稳定中。  
部分 API 与实现细节在后续迭代中仍可能发生变化。
