---

Proto UI does not expose a single component library. This section groups the current UI libraries that Proto UI reconstructs as prototype-driven outputs.

- `Base` is the protocol-first library closest to shared interaction semantics.
- `Shadcn` is a reference library that stays as close as possible to the upstream shadcn/ui authoring and interaction model.

Choose a library first, then browse its entries.
