The shadcn dialog family provides the styled building blocks for modal dialogs.

- **shadcn-dialog-root**: Holds the `open` state and provides context to all child parts.
- **shadcn-dialog-trigger**: Opens or toggles the dialog on click.
- **shadcn-dialog-overlay**: Renders the backdrop mask and follows the open state for enter/leave transitions.
- **shadcn-dialog-content**: Handles focus trapping, overlay dismissal strategies (ESC, outside press), and enter/leave transitions.
- **shadcn-dialog-title**: Semantic title automatically linked to content via `aria-labelledby`.
- **shadcn-dialog-description**: Semantic description automatically linked to content via `aria-describedby`.
- **shadcn-dialog-close**: Closes the dialog on click.

Pass `alert` to `shadcn-dialog-content` to turn it into an alert dialog: it disables outside-press dismissal while keeping ESC close.
